## Setup

### Shell

1. Clone the repo
2. `npm install`
3. `npm start`

### Docker

```shell
docker run mbround18/fvtt-dndbeyond-companion
```

### Kubernetes

```shell
helm repo add mbround18 https://mbround18.github.io/helm-charts/
```
